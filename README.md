# 數位分身小聚

用 AI 進入一座虛擬村莊，跟村民們聊天、辯論、深度討論。

每一個角色，背後都是一個真實的人。把你的思維模式、價值觀、判斷，訓練成你的 AI 分身，然後讓所有人的分身，在同一個村莊裡碰撞出新想法。

## 怎麼玩

1. 點右上角綠色按鈕 **Code** → **Download ZIP**，下載到電腦，解壓縮
2. 或者直接複製這個頁面的網址，貼給你的 AI 說「幫我下載這個」也行
3. 用 AI 工具開啟這個資料夾（例如 Cowork、Claude Code、Codex）
4. 打個招呼，村莊就會動起來

## 你可以做什麼

- 跟村民聊天：每個村民都有自己的個性、專長和口吻
- 辯論：說「辯論」，酒館掌櫃會接手主持
- 深度討論：說「深度討論」，茶館老闆帶大家往下挖
- 建立自己的數位分身：說「幫我建數位分身」
- 跟客座講師對話：納瓦爾老師、悉達多老師、陰陽哥都在村裡

## 這個專案在做什麼

數位分身小聚是一個實驗：讓每個人訓練自己的 AI 分身，然後把所有人的分身丟進同一個村莊聊天。

它表面上像遊戲，實際上在練四件事：

- AI 協作與系統設計
- 深度知識盤點
- 隱性知識提煉
- 連結不同維度的思維角度

整個系統的核心不是複雜程式，而是文字、資料夾、設定檔與技能包。村規負責世界運作，角色設定檔負責靜態知識，技能包負責特殊玩法與事件觸發。
