# 數位分身小聚

用 AI 進入一座虛擬村莊，跟村民們聊天、辯論、深度討論。

## 怎麼玩

1. 點右上角綠色按鈕 **Code** → **Download ZIP**，下載到電腦
2. 解壓縮
3. 用 AI 工具開啟這個資料夾（例如 Cowork、Claude Code、Codex）
4. 打個招呼，村莊就會動起來

## 你可以做什麼

- 跟村民聊天 — 每個村民都有自己的個性、專長和口吻
- 辯論 — 說「辯論」，酒館掌櫃會接手主持
- 深度討論 — 說「深度討論」，茶館老闆帶大家往下挖
- 建立自己的數位分身 — 說「幫我建數位分身」
- 跟客座講師對話 — 納瓦爾老師、悉達多老師、陰陽哥都在村裡

## 更新到最新版

如果你會用終端機，可以用 `git pull` 一鍵更新。
不會也沒關係，重新下載 ZIP 就是最新版。
