# 數位分身小聚 — Agent 下載版

用 AI Agent 開啟，就能進入初心小村，跟村民們一起聊天、辯論、深度討論。

## 快速開始

1. 下載或 clone 這個 repo
2. 用支援 CLAUDE.md 的 AI Agent 開啟這個資料夾（例如 Claude Code、Codex、Cowork）
3. 打個招呼，村莊就會動起來

```bash
git clone https://github.com/Jiang-Yude/digital-twin-gathering-agent.git
```

## 你可以做什麼

- 跟村民聊天 — 每個村民都有自己的個性、專長和口吻
- 辯論 — 說「辯論」，酒館掌櫃會接手主持，村民們正反交鋒
- 深度討論 — 說「深度討論」，茶館老闆會帶大家往下挖，異中求同、同中求異
- 建立自己的數位分身 — 說「幫我建數位分身」，跟著流程做出你的設定檔
- 跟客座講師對話 — 納瓦爾老師、悉達多老師、陰陽哥都在村裡

## 資料夾結構

```
CLAUDE.md                        <- AI 讀這個就能跑
02 世界觀、村規與風格設定/        <- 村莊規則
03 NPC角色設定設定檔/             <- 系統角色（村長、先知、教授等）
04 數位分身-個人設定檔/           <- 村民們的設定檔
05 NPC客座講師/                   <- 客座講師設定檔
_agent/skills/                    <- 技能包（辯論、深度討論、建設定檔等）
avatars/                          <- 角色大頭貼
```
