## 搜尋村民的具體步驟

1. 用 `grep_search` 在 `{ROOT}/04 數位分身-個人設定檔/` 中搜尋話題關鍵字
2. 從搜尋結果中找到相關的設定檔檔名
3. 用 `view_file` 讀取該設定檔的完整內容
4. 根據設定檔內容讓村民發言

搜尋客座講師時，在 `{ROOT}/05 NPC客座講師/` 中用同樣方式搜尋。

例外：醫學人體模型（中醫診所-001）是中醫診所場景專屬角色，設定檔存放在 `02 副本、特殊事件/中醫診所/中醫診所-001-人體模型-body_model.md`，不在 `05 NPC客座講師/`。
